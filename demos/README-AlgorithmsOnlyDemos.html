yFiles WPF Programming Samples Overview

The yFiles WPF programming samples are extensive Visual Studio projects with full C# source code that present the functionality of the yFiles WPF diagramming library. All programming samples can be found in subdirectories of the current directory. Most samples are self-contained, i.e., they consist of a single Visual Studio project.

Running a Sample

To open a specific programming sample, the corresponding Visual Studio project file (usually named [demoname].csproj) can be found in the demo's directory, which also uses the [demoname] naming scheme.
Note that samples that demonstrate related aspects are grouped together in solution folders. They also share a common prefix for their directory names and their namespaces.

All demos can also be accessed by opening the solution file (yFiles Demos.sln) in this folder.

Programming Samples Categories

Category Description
Algorithm Only Demos These samples show how to work with the graph structure and the analysis and layout algorithms in the algorithms package without using classes from the viewer part.
